My Alerts
Set alerts for the favorite route/stop combinations set on the "Favorites" panel. By selecting the Turn on alerts for all times option, all route and stop alerts from all route/stop combinations will be sent to by text message or email (see below), regardless of the day or time. The rest of the options allow you to set alerts for up to two time ranges over a full week, weekdays only, or weekends only. Finally, you can set the option to receive alerts via text message or email from the option bar below.
